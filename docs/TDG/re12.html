HTML_Progress2::getValue
Prev  Current value  Next


Name

HTML_Progress2::getValue - Returns the progress meter current value

Synopsis 

      require_once 'HTML/Progress2.php';

integer HTML_Progress2::getValue();
;

Description

Get the current value of the progress meter that is always between the minimum and maximum values, inclusive. The default value is equal to minimum.

Throws

throws no exceptions thrown

See

see HTML_Progress2::setValue , HTML_Progress2::incValue

Note

since 2.0.0

This function can not be called statically.
