Version 2.3.x

After two years of waiting, a PEAR user Request #3985, and help of PEAR::HTML_AJAX package, the AJAX progress bar is born.

  • This important version that add asynchronous progress bar, give also support of file upload, for both PHP4 and PHP5 users.

Asynchronous File Upload support is available with :

AFLAX (stands for Asynchronous Flash and XML)

The AJAX for Adobe Flash Platform 8 or greater. See official website.

PHP5 (5.2.0 or greater) extensions backend

APC that include RFC1867 fileupload processing hook (version 3.0.13 or greater).

uploadprogress that used the RFC1867 hook to track progress of file upload.

PHP patches

Available for versions 4.3.10 and 5.0.3
