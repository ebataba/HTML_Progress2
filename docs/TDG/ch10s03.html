What was removed ?

Very few features was removed during this merges :

  • setModel() method.
  • ability to extends UI class to make custom user interface.
  • ability to extends DM class to make custom data management interface.

And some replacements was made with removes of :

  • isStringPainted() method (due to new multiple label system).
  • setStringPainted() method (due to new multiple label system).
  • setStringAttributes() method, replaced by setLabelAttributes().
  • getString() method (due to new multiple label system).
  • setString() method (due to new multiple label system).
  • HTML_PROGRESS_ERRORSTACK_* and HTML_PROGRESS_LOG_TYPE_* constants (due to new error management system).
