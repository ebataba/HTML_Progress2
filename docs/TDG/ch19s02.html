Listener

HTML_Progress2::addListener - Attachs a new observer
HTML_Progress2::removeListener - Removes a registered observer.

addListener() and removeListener() methods provides an implementation of the observer pattern. In the content of the HTML_Progress2 package, it provides a mechanism by which you can examine special event as it is happened. This allows the implementation of special behavior.
